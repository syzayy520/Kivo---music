# Home Design Iteration 001

## What Was Kept

- Left navigation and persistent transport bar from the source design.
- Home-first layout with recommendation, resume cards, library additions, and
  recent listens.
- Light native desktop mood.

## What Was Improved

- Added evidence-backed audio truth to the player bar.
- Made the left navigation denser and calmer.
- Reduced decorative card nesting and kept repeated content cards purposeful.
- Added a right-side device/truth area to make Kivo feel like a serious audio
  product rather than a generic streaming clone.
- Split implementation into small QML component families.

## Next Polish Loop

- Replace generated artwork blocks with real design assets.
- Add focus, hover, and keyboard states once Qt rendering can be verified.
- Connect mock view state to a C++ view model before Host ABI integration.
